# css-greeting-card
A responsive birthday greeting card built using HTML and CSS. This project demonstrates the use of CSS pseudo-elements, hover effects, transitions, transforms, flexbox, and the :target pseudo-class to create a dynamic user experience. Users can send the card or share it on social media through clickable links that reveal hidden sections.
# 🎉 Greeting Card Project

A simple and interactive Birthday Greeting Card created with HTML and CSS.

## ✨ Features

- 🎂 Beautiful birthday card design
- 🥳 Emoji decorations using `::before` and `::after`
- 🎨 Hover effects with scaling and color transitions
- 🔗 Interactive links for "Send Card" and "Share on Social Media"
- 🎯 Hidden sections revealed using the `:target` pseudo-class
- 📐 Flexbox layout for button alignment
- 🌀 Transform effects with `scale()` and `skewX()`

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📚 CSS Concepts Practiced

- `transition`
- `transform`
- `:hover`
- `:active`
- `:focus`
- `:visited`
- `::before`
- `::after`
- '::target'
